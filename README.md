# élevé — Premium Hair Studio

Лендинг премиальной студии ухода за волосами.

## Структура
- `index.html` — одностраничный сайт (весь код в одном файле)
- `assets/` — изображения (4K-фотографии работ, интерьер, фоны)

## Технологии
- HTML + Tailwind CSS (через CDN)
- Ванильный JavaScript
- Анимации: blur-fade при скролле, marquee, интерактивный слайдер «До/После», параллакс, счётчики, кастомный курсор
- Шрифты: Cormorant Garamond (заголовки) + Inter (текст)

## Локальный запуск
Просто откройте `index.html` в браузере — сборка и сервер не нужны.

## Деплой
Статический сайт. На Vercel разворачивается из корня репозитория без настроек сборки.
